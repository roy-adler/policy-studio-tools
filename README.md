# Policy Studio Tools

VS Code extension for Axway Policy Studio projects: multi-project discovery, circuit search, and a Policy Studio sidebar.

## Features

- Discover Policy Studio projects in monorepos
- Search circuits, filters, attributes, scripts, and references
- Policy Studio sidebar with project scope and tools

## Requirements

- VS Code 1.85 or newer

## Development

```bash
npm ci
npm test
npm run compile
```
